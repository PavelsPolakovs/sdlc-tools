# sdlc-tools

Монорепозиторий с MCP-серверами для SDLC-пайплайнов.

## Пакеты

- [`packages/sdlc-ship-changes`](packages/sdlc-ship-changes) — MCP-сервер, который выставляет пайплайн "ship changes" (Jira + GitLab) как курируемый набор MCP-инструментов.

Гайд по разработке и архитектуре — в [`CLAUDE.md`](CLAUDE.md).
